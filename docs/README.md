# FoundryUI

FoundryUI is a visual workflow interface for protein and ligand design workflows. It connects typed nodes for loading structures, selecting residues or atoms, running Foundry tools, filtering results, viewing structures, and saving artifacts.

The workflow canvas is built around three ideas:

- <span class="type protein">Nodes</span> are operations.
- <span class="type sequence">Types</span> describe what can flow between operations.
- <span class="type score">Runs</span> execute the graph and preserve intermediate artifacts.

Start with [Visual programming](principles/visual-programming.md), then read [Nodes](principles/concepts/nodes.md) and [Types](principles/concepts/types.md).
