# Residue Atom Selector

Manual node that selects atoms inside selected protein residues.

Inputs:

- `List of Residues`: <span class="type residue">List of Residues</span>.
- `Protein`: <span class="type protein">Protein</span>.

Options:

- `Protein atoms`: JSON-like mapping such as `{"A56":"CG,OH","A115":"CG,SD"}`.
- `Open`: viewer that displays selected residues in sticks mode.

Output:

- `proteinAtoms`: <span class="type residue-atoms">Residues Atoms List</span>.
