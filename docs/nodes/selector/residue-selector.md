# Residue Selector

Manual node that emits <span class="type residue">List of Residues</span>.

Inputs:

- `Protein*`: optional upstream <span class="type protein">Protein</span>.

Options:

- `Residues`: comma-separated residue IDs such as `A58,A103`.
- `Open`: runtime viewer.

Output:

- `residues`: residue IDs as <span class="type residue">List of Residues</span>.
