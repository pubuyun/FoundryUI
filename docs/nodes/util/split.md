# Split

Splits complex PDB files into ligand and protein batches.

> [!input]
- `Batch Protein (With Ligand)`: <span class="type complex">Batch Protein (With Ligand)</span>.

> [!output]
- `batchLigand`: <span class="type batch-ligand">Batch Ligand</span>.
- `batchProtein`: <span class="type batch-protein">Batch Protein</span>.
